# Express-typescript-postgres

## Getting Started


## Prerequisites
- Make sure install docker, docker-compose
- Node version: 16lts
- Yarn version 3.2.2

## Installation
- Run docker postgres: 

```
docker-compose pg up -d
```

- Install NPM packages
```
npm i
```

- Run dev
```
npm run dev
```

- Run start
```
npm run build:tsc
npm run build
npm run start
```
